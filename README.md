# Pokedex

Pokemons catalog 
>Front-end with attribute details, filter, autocomplete and favorite.
>
>Back-end integrated with dedicated database and request filter

## Lib used
### Front-End
+ React with Vite
+ React Router Dom
+ Context with Hook useContext
+ Styled-Components
+ Mui Material

### Back-End
+ Express
+ Mongoose
+ Cors
+ Dotenv

### Database
+ MongoDB

## Install

1. Clone the application.
2. Open the terminal and type `npm install` to install the dependencies.
3. Then upload the app with `npm run dev`.
